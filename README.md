# QA Pramaan Automation Project

This project automates workflows on `https://qa-pramaan.vercel.app/` using Python and Playwright.

## Setup Instructions

We highly recommend opening this project folder directly in your IDE as your active workspace:
`C:\Users\Admin\.gemini\antigravity\scratch\qa-pramaan-automation`

### 1. Create and Activate Virtual Environment
Open your terminal located in this folder and run:
```powershell
python -m venv venv
.\venv\Scripts\activate
```

### 2. Install Dependencies
```powershell
pip install -r requirements.txt
playwright install chromium
```

### 3. Run the Code!
```powershell
python automation.py
```
